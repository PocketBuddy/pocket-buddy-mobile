# PocketBuddy

Based on [The React Native Boilerplate](https://github.com/thecodingmachine/react-native-boilerplate)

## Dependencies

| Dependency | Version |
| ---------- | ------- |
| Node       | 18.15.0 |
| Yarn       | 1.22.19 |

## Development

First create .env file

`cp template.env .env`

Install dependencies

`yarn && yarn pod-install`

Run Metro

`yarn start`

And open your emulator device

`yarn ios | android`
